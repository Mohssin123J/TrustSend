## About The Project

TrustSend is a modern digital financial platform inspired by leading fintech companies such as Wise, Revolut and Payoneer.

The platform enables users to perform secure financial operations through an intuitive and professional interface while maintaining high levels of security and performance.

### Main Services

* National Money Transfers
* International Money Transfers
* Bill Payments
* Mobile Recharge
* Card Request & Management
* Transaction Tracking
* PDF Receipt Generation
* Complaint Management
* SMS Verification
* User Dashboard
* Administrator Dashboard

---

## Features

### Secure Transfers

Send money quickly and securely to local and international beneficiaries.

### Bill Payments

Pay utility bills, subscriptions and services from a single platform.

### Smart Dashboard

Monitor balances, transactions and activities in real time.

### Card Management

Request, activate, deactivate or replace cards directly from the dashboard.

### PDF Receipts
